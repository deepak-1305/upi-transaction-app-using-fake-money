# upi-transaction-app-using-fake-money
The Fake UPI Transaction App is a React Native–based mobile application designed to simulate the experience of digital payments in India’s UPI ecosystem. Instead of real currency, the app uses virtual/fake money to provide a safe, interactive environment for learning, testing, and demonstrating UPI-style transactions.

🎯 Purpose
To mimic the look and feel of real UPI apps like Google Pay or PhonePe.

To provide a sandbox environment for experimenting with digital transactions without financial risk.

To serve as a learning tool for students, developers, and fintech enthusiasts exploring mobile payment systems.

⚙️ Key Features
Dashboard Screen: Displays balance, transaction history, and analytics.

Send/Receive Money Simulation: Users can transfer fake currency between accounts.

Transaction History: Logs all simulated payments with timestamps and details.

Analytics & Charts: Visual representation of spending and transfers using animated graphs.

Multi-Screen Navigation: Smooth transitions between Dashboard, Send Money, and History screens.

UI/UX Design: Vibrant, colorful, 3D-inspired interface with neon effects, inspired by Google Pay’s polished design.
